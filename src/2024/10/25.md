## 6. 为什么 uint8_t 取反会变成 int

```cpp
#include <cstdint>
#include <type_traits>
int main() {
    uint8_t x = 0;
    static_assert(std::is_same_v<int, decltype(~x)>);
}
```

算术运算符都会整数提升。

## 7. Java 能不能鸭子类型

普通的继承：声明的时候得写明它继承了 Duck 类。

鸭子类型：只要这个类实现了 fly()，它就是鸭子。

Go 和动态类型语言（Python, JS）都是采用鸭子类型。（C++ 的 proxy 库可能也是？）

Java 可以通过反射来完成。
