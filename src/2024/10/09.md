## 1. Linux fopen 一个文件，使用过程中文件被删了

文件的 use count 变成 0 才会真正删除（释放磁盘空间），fopen 可以继续使用。

## 2. C29 named break / continue

C29 新增语法可以跳出多重循环等操作。相比于 goto 有使用上的限制。

```c
outer:
for (int i = 0; i < 10; i++) {
    for (int j = 0; j < 10; j++) {
        break outer;
    }
}
```
