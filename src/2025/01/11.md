## 2. 返回值协变

子类重写父类方法时，返回值可以是父类返回值的子类。

```cpp
class Base {
public:
    virtual Base* clone() const {
        return new Base(*this);
    }
};
 
class Derived : public Base {
public:
    virtual Derived* clone() const override {
        return new Derived(*this);
    }
};
```

## 3. 开源是什么模式，为什么公司会雇人写开源
